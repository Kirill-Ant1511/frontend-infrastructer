# AGENT.md - инструкция для ИИ-ассистента

## Обзор проекта

- **Стек:** React 19, JavaScript, Vite, CSS
- **Цель:** Простой задачник для отслеживания задач
- **Архитектура:** Чистая архитектура с разделением на слои (domain, application, infrastructure)

## Команды

- Установка: `yarn install`
- Запуск: `yarn dev`
- Сборка: `yarn run build`
- Линтинг: `yarn lint`
- Форматер: `yarn format`

## Стиль кода и паттерны

- **TypeScript:** Строгий режим (strict mode).
- **Форматирование:** Одинарные кавычки, c точками с запятой (semicolons: true).
- **Компоненты:** Только функциональные компоненты на хуках.
