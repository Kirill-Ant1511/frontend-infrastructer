# hh-homework-js

Четыре задания по JavaScript, каждое в отдельном файле:

1. `task1-promise-all.js` — реализуйте свой `promiseAll`
2. `task2-delay.js` — реализуйте `delay(ms)`
3. `task3-memoize.js` — реализуйте `memoize(fn)` (аргументы — строки/числа)
4. `task4-typed-object.js` — реализуйте `typedObject(schema)`

Процесс:
- Клонируете репозиторий
- Создаете свою ветку (называете как-то так, чтобы не совпало с другими, например фамилия_имя)
- Выполняете задания в указанных файлах
- Делаете пулл-реквест обратно в этот репозиторий
